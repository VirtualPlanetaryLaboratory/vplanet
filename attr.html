Attribution¶

After your first pull request to the code has been accepted, you are officially part of the collaboration! As such you will be named in the Author List, and be included as a co-author in future publications about the source code. Note that contributors can request anonymity and that failure to adhere to the Code of Conduct can result in expulsion from the collaboration.
